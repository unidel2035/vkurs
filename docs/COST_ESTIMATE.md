## РИСКИ И ДОПУЩЕНИЯ

### Риски

| Риск | Вероятность | Влияние | Митигация |
|------|-------------|---------|-----------|
| Изменение API Claude | Средняя | Высокое | Абстракция сервиса, fallback |
| Изменение API hh.ru | Низкая | Среднее | Graceful degradation |
| Увеличение scope | Высокая | Высокое | Change request процедура |
| Недоступность Integram | Низкая | Высокое | Резервное решение |

### Допущения

1. Заказчик предоставляет доступ к Integram
2. Заказчик оплачивает API Claude (или возмещает расходы)
3. Требования зафиксированы в ТЗ
4. Изменения scope оформляются как Change Request с пересчётом
