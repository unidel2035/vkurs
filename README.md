# Верный Курс

**AI-powered чат-бот для профориентации молодых людей**

## Описание проекта

«Верный Курс» — это интеллектуальный помощник, который помогает молодым людям найти профессию по душе. Система использует ценностный подход к профориентации: выявляет ключевые жизненные ценности пользователя через диалог и подбирает профессии, максимально соответствующие его внутренним приоритетам.

### Ключевые возможности

- **Диалоговое выявление ценностей** — AI-ассистент в формате естественного разговора определяет жизненные приоритеты пользователя
- **Подбор профессий по ценностям** — каждая профессия в базе классифицирована по 12 ключевым ценностям
- **Детальное описание профессий** — подробная информация о выбранных специальностях
- **Поиск вакансий** — интеграция с hh.ru для поиска актуальных предложений
- **Анализ компетенций** — оценка необходимых навыков и текущего уровня пользователя
- **Рекомендации по обучению** — подбор учебных центров, курсов и стажировок
- **Персональный план развития** — пошаговая дорожная карта к желаемой профессии

## Технологический стек

### Frontend
- **Vue 3** — прогрессивный JavaScript-фреймворк
- **PrimeVue** — UI-библиотека компонентов
- **Pinia** — управление состоянием
- **Vue Router** — маршрутизация

### Backend & Data
- **Integram** — low-code платформа для хранения данных и бизнес-логики
- **Claude API** — AI-движок для диалогов и анализа

### Интеграции
- **hh.ru API** — поиск вакансий
- **Образовательные платформы** — курсы и стажировки

## Модель ценностей

Система использует 12 ключевых жизненных ценностей для классификации профессий:

| Ценность | Описание |
|----------|----------|
| **Материальный достаток** | Высокий доход, финансовая стабильность |
| **Карьерный рост** | Продвижение, статус, влияние |
| **Творчество** | Креативность, самовыражение |
| **Помощь людям** | Социальная значимость, альтруизм |
| **Стабильность** | Предсказуемость, надёжность |
| **Независимость** | Автономность, свобода решений |
| **Признание** | Уважение, известность |
| **Баланс** | Гармония работы и личной жизни |
| **Интеллектуальное развитие** | Обучение, сложные задачи |
| **Командная работа** | Сотрудничество, коллектив |
| **Инновации** | Новые технологии, прогресс |
| **Практический результат** | Осязаемые достижения |

## Путь пользователя

```
1. Знакомство
   └── AI приветствует и объясняет процесс

2. Диалог о ценностях (10-15 минут)
   └── Серия открытых вопросов о жизни, интересах, мечтах
   └── AI выявляет топ-5 ценностей пользователя

3. Результаты анализа
   └── Визуализация профиля ценностей
   └── Список подходящих профессий с % совпадения

4. Исследование профессий
   └── Детальное описание каждой профессии
   └── Необходимые компетенции
   └── Средняя зарплата и перспективы

5. Поиск возможностей
   └── Актуальные вакансии с hh.ru
   └── Анализ требований vs навыки пользователя

6. План развития
   └── Рекомендации по обучению
   └── Стажировки от компаний
   └── Пошаговая дорожная карта

7. Трекинг прогресса
   └── Отслеживание выполнения плана
   └── Корректировка рекомендаций
```

## Быстрый старт

### Требования
- Node.js 18+
- Bun (рекомендуется) или npm

### Установка

```bash
# Клонировать репозиторий
git clone https://github.com/unidel2035/vkurs.git
cd vkurs

# Установить зависимости
bun install

# Настроить переменные окружения
cp .env.example .env
# Отредактировать .env

# Запустить в режиме разработки
bun dev
```

### Переменные окружения

```env
VITE_INTEGRAM_URL=https://app.integram.io
VITE_INTEGRAM_DB=vkurs
VITE_CLAUDE_API_KEY=your_api_key
VITE_HH_API_KEY=your_hh_api_key
```
